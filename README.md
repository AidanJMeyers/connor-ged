# Connor's GED Study Dashboard

Interactive, comprehensive GED preparation dashboard covering all four subject areas:

- **Mathematical Reasoning** — Number sense, fractions/decimals, ratios/percents, data & probability, algebra, geometry (with full step-by-step solutions for every concept)
- **Reasoning Through Language Arts (RLA)** — Reading comprehension, fiction, writing, grammar, mechanics, extended response
- **Social Studies** — U.S. history, civics & government, economics, world geography
- **Science** — Life science, Earth & space, physical science, scientific reasoning

## Features

- 25 chapters with detailed study blocks
- Per-chapter practice questions with explanations
- 100-question comprehensive practice exam
- Reviewed checkbox + 1–5 confidence rating per block
- Comment boxes on every block & question, surfaced in the **Review Later** tab
- Markdown export of all flagged notes
- Persistent progress (localStorage / `window.storage`)
- Mobile-responsive
